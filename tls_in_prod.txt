"Step 1) 
create wallet to configure TLS "	"Make Directory to create wallet

mkdir -p /u01/app/grid/admin/wallet

orapki wallet create -wallet <wallet location> -auto_login -pwd <password>

E.g: orapki wallet create -wallet /u01/app/grid/admin/wallet -pwd Password#1 -auto_login

Note:- 1) this wallet to be created under GRID HOME  "
"Step 2) 
Modify listeners"	"1) verify Endpoints

crsctl stat res -p |grep ENDPOINTS

2)Add New port fot TLS

srvctl modify listener -p ""TCP:1787/TCPS:1789""

3)Take Listner Restart 

srvctl stop listener 
srvctl start listener

4)Adding  TCPS endpoints to scan listeners

 srvctl stop scan_listener
 
 srvctl modify scan_listener -p TCP:1786/TCPS:1788
 
 srvctl start scan_listener
 
5) verify that endpoint have been added in scan listener 

crsctl stat res -p |grep ENDPOINTS"

"Step 4)

Define wallet locations in listener.ora and sqlnet.ora"	"Note:- This changes to be done in grid Home 
Note:- Add these entry in listener.ora & sqlnet.ora  in each node 

a) SQLNET.ORA

1) Take backup of sqlnet.ora and listner.ora

cp sqlnet.ora sqlnet_bkp_13062025
cp listener.ora listener_bkp_13062025


2) Add below entry in sqlnet.ora


cat  $ORACLE_HOME/network/admin/sqlnet.ora 

SQLNET.AUTHENTICATION_SERVICES= (BEQ, TCPS)

SSL_CLIENT_AUTHENTICATION = FALSE

WALLET_LOCATION =
  (SOURCE =
    (METHOD = FILE)
    (METHOD_DATA =
      (DIRECTORY = /u01/app/grid/admin/wallet)
    )
  )


b)listener.ora

1) Add below entry in listener.ora

SL_CLIENT_AUTHENTICATION = FALSE

WALLET_LOCATION =
  (SOURCE =
    (METHOD = FILE)
    (METHOD_DATA =
      (DIRECTORY = /u01/app/grid/admin/wallet)
    )
  )"
"Step 5)
Testing connection"	"Add below entry in TNSNAMES.ORA
DBSSL =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCPS)(HOST = eissipr-scan1:1788 )(PORT =1788))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = <service name>)
    )
  )


sqlnet sys@DBSSL as sysdba"
